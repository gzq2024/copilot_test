Yolo V8を利用してヘルメット着用検出

１．Kaggle　から　「Construction Site Safety Image Dataset Roboflow」をダウンロード
２．環境作成
　　Python仮想環境作り：py -3.10 -m venv yolo8_safehat
　　環境活性化：E:\Project\Yolo\yolo8_safehat\Scripts\activate
　　Yolo v8インストール：pip install ultralytics
３．yamlファイル作成
　　safehat.yamlを作成し、「E:\Project\Yolo\yolo8_safehat\Lib\site-packages\ultralytics\cfg\models\v8」に格納する。
４．訓練ソース作成
　　yolo_train.py
５．訓練実施
       python yolo_train.py
      ※訓練課程データおよびモデルデータは、「\runs\detect\train*\」配下に格納される。※
　　 weightsにモデルデータが格納される
６．テスト実施
　　「train5\best.pt」を利用して、テストを実施
      テスト時、識別したいクラスのみを指定することが可能。
　　識別結果は、「\runs\detect\predict」に格納される。
